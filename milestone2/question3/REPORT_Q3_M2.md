## Question 3

### Question itself
Is provincial labour-market pressure (job vacancy rate) associated with voter participation quality (turnout and rejected-ballot rate) in the 2019 and 2021 federal elections?

Question status for Milestone II: unchanged from Milestone I.

### Preliminary data processing required

#### Source files used
- 2019 election outcomes: `election43/table_tableau11.csv`
- 2021 election outcomes: `election44/table_tableau11.csv`
- Statistics Canada job vacancy table 14-10-0442-01: `statscan/14100442.csv`

#### Fields used
- Election fields: `Province`, `Electors`, `Valid Ballots`, `Rejected Ballots`, `Total Ballots Cast`
- Vacancy fields: `REF_DATE`, `GEO`, `North American Industry Classification System (NAICS)`, `Statistics`, `VALUE`

#### Selections and summarizations performed
1. Election preprocessing:
- Normalize province names (bilingual province labels reduced to consistent English key).
- Aggregate riding-level rows to province-year totals.
- Compute participation metrics from totals:
  - `turnout = total_ballots / electors * 100`
  - `valid_rate = valid_ballots / total_ballots * 100`
  - `rejected_rate = rejected_ballots / total_ballots * 100`

2. Vacancy preprocessing:
- Filter StatsCan rows to `Statistics = Job vacancy rate`.
- Exclude `GEO = Canada` to keep provincial/territorial level analysis.
- Keep NAICS sector as a parameterized dimension.
- Compute election-window averages for each province-year-sector:
  - `quarters_before = 1, 2, 4`
  - 2019 window anchored before October 2019 election date.
  - 2021 window anchored before September 2021 election date.

3. Join and final encoding:
- Join election aggregates with vacancy windows on `province + year`.
- Repeat election metrics across sectors/windows to produce analysis-ready matrix.

#### Final encodings and organization into new files
Generated by `scripts/q3_preprocess.py`:
- `milestone2/question3/data/processed/q3_election_summary.csv`
  - one row per province-year with ballot totals and rates.
- `milestone2/question3/data/processed/q3_vacancy_windows.csv`
  - one row per province-year-sector-window with `vacancy_avg`.
- `milestone2/question3/data/processed/q3_analysis_matrix.csv`
  - one row per province-year-sector-window with both vacancy and participation metrics.

### Parameters for this question
Used by `scripts/q3_demo.py`:
- `--year {2019,2021}`
- `--province {ALL, <province-name>}`
- `--sector {NAICS sector label}`
- `--metric {turnout,rejected_rate,valid_rate}`
- `--quarters-before {1,2,4}`

### Final data processing during demo time
- Load `q3_analysis_matrix.csv`.
- Filter rows by `sector` and `quarters_before`.
- If `--province ALL`:
  - keep selected `year`.
  - output province-level table and Pearson correlation between `vacancy_avg` and selected metric.
  - generate scatter plot SVG.
- If a single province is selected:
  - use 2019 and 2021 rows for that province.
  - generate two-year comparison bar chart SVG for selected metric.
